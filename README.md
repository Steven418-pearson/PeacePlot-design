# PeacePlot AI — design bundle

Static HTML/CSS design prototypes for PeacePlot.

## Folders

- **`blue-white/`** — Primary landing (`index.html`), auth (`sign-in.html`, `sign-up.html`, `auth.css`), search. See `blue-white/LANDING_PLAN.md` and `blue-white/AUTH_PLAN.md`.
- **`purple/`** — Alternate theme exploration.

Open `blue-white/index.html` in a browser from that directory so asset paths resolve.
